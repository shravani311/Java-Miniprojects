# 📚 Java Mini Projects Collection

This repository contains a collection of basic Java mini projects developed for practice and understanding core programming concepts.

---

## 🚀 Projects Included

### 1. 📞 Contact Management System

* Files: `contactmanagement.java`, `contact.class`
* Features:

  * Add new contacts
  * Store contact details
  * Basic console-based interaction

---

### 2. 📊 Grade System

* Files: `gradesys.java`, `gradesys.class`
* Features:

  * Calculate grades based on marks
  * Simple grading logic implementation

---

### 3. 🔤 Character Frequency Counter

* Files: `frequency.java`, `frequency.class`
* Features:

  * Counts frequency of characters in a string
  * Uses HashMap for efficient counting

---

### 4. 📚 Library Management System

* Files: `libmanagement.java`, `library.class`
* Features:

  * Add books
  * Manage library records
  * Basic operations using Java classes

---

## 🛠️ Technologies Used

* Java (Core Java)
* OOP Concepts
* Collections Framework (HashMap, ArrayList)

---

## ▶️ How to Run

1. Compile the Java file:

   ```bash
   javac filename.java
   ```

2. Run the program:

   ```bash
   java classname
   ```

### Example:

```bash
javac contactmanagement.java
java contactmanagement
```

---

## ⚠️ Note

* `.class` files are compiled files and are usually not uploaded to GitHub.
* It is recommended to add `.class` files to `.gitignore`.
